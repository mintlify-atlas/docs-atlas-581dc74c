---
title: 'Logout'
api: 'GET /app/logout'
description: 'Logout from WhatsApp and remove all session data from the server'
---

Remove the device session from WhatsApp and delete all associated database files.

## Authentication

<ParamField header="Authorization" type="string" required>
  Basic authentication credentials (username:password in base64)
</ParamField>

## Headers

<ParamField header="X-Device-Id" type="string">
  Device identifier for multi-device support. Required when multiple devices are registered. If only one device is registered, it will be used as the default.
</ParamField>

## Response

<ResponseField name="code" type="string">
  Response code indicating success or failure
  
  **Example:** `SUCCESS`
</ResponseField>

<ResponseField name="message" type="string">
  Human-readable response message
  
  **Example:** `Success`
</ResponseField>

<ResponseField name="results" type="string">
  Additional result data (typically null)
  
  **Example:** `null`
</ResponseField>

## Example Request

<CodeGroup>

```bash cURL
curl -X GET 'http://localhost:3000/app/logout' \
  -u 'username:password' \
  -H 'X-Device-Id: my-device-id'
```

```javascript JavaScript
const response = await fetch('http://localhost:3000/app/logout', {
  method: 'GET',
  headers: {
    'Authorization': 'Basic ' + btoa('username:password'),
    'X-Device-Id': 'my-device-id'
  }
});

const data = await response.json();
console.log(data);
```

```python Python
import requests
import base64

username = 'username'
password = 'password'
credentials = base64.b64encode(f'{username}:{password}'.encode()).decode()

response = requests.get(
    'http://localhost:3000/app/logout',
    headers={
        'Authorization': f'Basic {credentials}',
        'X-Device-Id': 'my-device-id'
    }
)

print(response.json())
```

</CodeGroup>

## Response Example

```json
{

```

## Usage Notes

- This operation is **irreversible** - all session data will be permanently deleted
- After logout, you will need to login again using QR code or pairing code
- The device will be removed from your WhatsApp linked devices list
- All local message history and chat data will be deleted from the server
