---
title: 'Create Group'
openapi: 'POST /group'
---

Create a new WhatsApp group with specified participants.

<Info>
You must include at least one participant to create a group. You will automatically be added as the group admin.
</Info>

## Request

<ParamField header="X-Device-Id" type="string">
  Device identifier for multi-device support. Required when multiple devices are registered. If only one device is registered, it will be used as the default.
</ParamField>

<ParamField body="title" type="string" required>
  The name of the group to create
  
  Example: `Example Group Title`
</ParamField>

<ParamField body="participants" type="array" required>
  Array of phone numbers (without country code suffix) to add as group members
  
  Example: `["6819241294719274", "6829241294719274", "6839241294719274"]`
</ParamField>

## Response

<ResponseField name="code" type="string">
  Response code (e.g., `SUCCESS`)
</ResponseField>

<ResponseField name="message" type="string">
  Response message describing the result
  
  Example: `Success get list groups`
</ResponseField>

<ResponseField name="results" type="object">
  <ResponseField name="group_id" type="string">
    The unique identifier of the newly created group
    
    Example: `1203632782168851111@g.us`
  </ResponseField>
</ResponseField>

<RequestExample>
```bash cURL
curl -X POST \
  'https://your-api-domain.com/group' \
  -H 'Content-Type: application/json' \
  -H 'X-Device-Id: my-device' \
  -u 'username:password' \
  -d '{
    "title": "Example Group Title",
    "participants": [
      "6819241294719274",
      "6829241294719274",
      "6839241294719274"
    ]
  }'
```
</RequestExample>

<ResponseExample>
```json Response
{
  "code": "SUCCESS",
  "message": "Success get list groups",
  "results": {
    "group_id": "1203632782168851111@g.us"
  }
}
```
</ResponseExample>
