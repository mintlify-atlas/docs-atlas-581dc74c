---
title: "Remove Device"
api: "DELETE /devices/{device_id}"
description: "Remove a device from the server (does not logout from WhatsApp)"
---

Remove a device from the server's device registry. Note that this operation only removes the device from the server - it does not log out the device from WhatsApp. To logout from WhatsApp, use the [Logout Device](/api/devices/logout) endpoint first.

## Endpoint

```
DELETE /devices/{device_id}
```

## Authentication

This endpoint requires HTTP Basic Authentication.

## Example Request

<CodeGroup>
```bash cURL
curl -X DELETE http://localhost:3000/devices/my-device-id \
  -u username:password
```

```javascript JavaScript
const deviceId = 'my-device-id';

const response = await fetch(`http://localhost:3000/devices/${deviceId}`, {
  method: 'DELETE',
  headers: {
    'Authorization': 'Basic ' + btoa('username:password')
  }
});

const data = await response.json();
console.log(data);
```

```python Python
import requests
from requests.auth import HTTPBasicAuth

device_id = 'my-device-id'

response = requests.delete(
    f'http://localhost:3000/devices/{device_id}',
    auth=HTTPBasicAuth('username', 'password')
)

print(response.json())
```

```go Go
package main

import (
    "fmt"
    "io"
    "net/http"
)

func main() {
    deviceID := "my-device-id"
    
    client := &http.Client{}
    req, _ := http.NewRequest("DELETE", "http://localhost:3000/devices/"+deviceID, nil)
    req.SetBasicAuth("username", "password")
    
    resp, _ := client.Do(req)
    defer resp.Body.Close()
    
    body, _ := io.ReadAll(resp.Body)
    fmt.Println(string(body))
}
```

## Important Notes

<Warning>
  **This endpoint does NOT logout the device from WhatsApp.** It only removes the device from the server's registry. 
  
  To properly logout a device from WhatsApp before removal:
  1. First call [POST /devices/{device_id}/logout](/api/devices/logout)
  2. Then call this endpoint to remove the device from the server
</Warning>

<Info>
  If you remove a device that is still logged into WhatsApp without logging out first, the device will remain active on WhatsApp but won't be manageable through this API anymore.
</Info>

## Use Cases

- Clean up unused device slots
- Remove devices after proper logout
- Manage device inventory in multi-device setups
- Free up resources by removing inactive devices
