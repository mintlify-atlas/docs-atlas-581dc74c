---
title: "Add Device"
api: "POST /devices"
description: "Create a new device slot for multi-device management"
---

Create a new device slot that can be used to connect a WhatsApp account. You can optionally provide a custom device ID, or let the system generate one automatically.

## Endpoint

```
POST /devices
```

## Authentication

This endpoint requires HTTP Basic Authentication.

## Request Body

<ParamField body="device_id" type="string" optional>
  Optional custom device ID. If not provided, a UUID will be generated automatically.
  <Expandable title="example">
    ```
    my-custom-device-id
    ```
  </Expandable>
</ParamField>

## Request Example

```json
{
  "device_id": "my-custom-device-id"
}
```

Or for auto-generated device ID:

```json
{}
```

## Example Request

<CodeGroup>
```bash cURL
curl -X POST http://localhost:3000/devices \
  -u username:password \
  -H "Content-Type: application/json" \
  -d '{"device_id": "my-custom-device-id"}'
```

```bash cURL (Auto-generate ID)
curl -X POST http://localhost:3000/devices \
  -u username:password \
  -H "Content-Type: application/json" \
  -d '{}'
```

```javascript JavaScript
const response = await fetch('http://localhost:3000/devices', {
  method: 'POST',
  headers: {
    'Authorization': 'Basic ' + btoa('username:password'),
    'Content-Type': 'application/json'
  },
  body: JSON.stringify({
    device_id: 'my-custom-device-id'
  })
});

const data = await response.json();
console.log(data);
```

```python Python
import requests
from requests.auth import HTTPBasicAuth

response = requests.post(
    'http://localhost:3000/devices',
    auth=HTTPBasicAuth('username', 'password'),
    json={'device_id': 'my-custom-device-id'}
)

print(response.json())
```

```go Go
package main

import (
    "bytes"
    "encoding/json"
    "fmt"
    "io"
    "net/http"
)

func main() {
    payload := map[string]string{"device_id": "my-custom-device-id"}
    jsonData, _ := json.Marshal(payload)
    
    client := &http.Client{}
    req, _ := http.NewRequest("POST", "http://localhost:3000/devices", bytes.NewBuffer(jsonData))
    req.SetBasicAuth("username", "password")
    req.Header.Set("Content-Type", "application/json")
    
    resp, _ := client.Do(req)
    defer resp.Body.Close()
    
    body, _ := io.ReadAll(resp.Body)
    fmt.Println(string(body))
}
```
</CodeGroup>

   - [Pairing Code Login](/api/devices/login)
2. Use the device ID in subsequent API calls via the `X-Device-Id` header
