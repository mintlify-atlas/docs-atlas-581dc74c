<div className="mt-16 mb-16 max-w-5xl mx-auto px-6">
  <h2 className="text-2xl sm:text-3xl font-bold text-gray-900 dark:text-white mb-4">
    Quick start
  </h2>
  <p className="text-base text-gray-600 dark:text-gray-400 mb-8">
    Get up and running with GOWA in minutes
  </p>

  <Steps>
    <Step title="Install and run the server">
      Choose your preferred installation method:

      <Tabs>
        <Tab title="Docker">
          ```bash
          docker run -d -p 3000:3000 \
            --name whatsapp \
            --restart always \
            -v whatsapp:/app/storages \
            aldinokemal2104/go-whatsapp-web-multidevice rest
          ```
        </Tab>
        <Tab title="Docker Compose">
          Create a `docker-compose.yml` file:
          ```yaml
          services:
            whatsapp:
              image: aldinokemal2104/go-whatsapp-web-multidevice
              container_name: whatsapp
              restart: always
              ports:
                - "3000:3000"
              volumes:
                - whatsapp:/app/storages
              command:
                - rest
          volumes:
            whatsapp:
          ```
          Then run:
          ```bash
          docker-compose up -d
          ```
        </Tab>
        <Tab title="Binary">
          Download the binary from [GitHub Releases](https://github.com/aldinokemal/go-whatsapp-web-multidevice/releases/latest):
          ```bash
          # Linux/macOS
          ./whatsapp rest

          # Windows
          .\whatsapp.exe rest
          ```
        </Tab>
      </Tabs>

      The server will start on `http://localhost:3000`
    </Step>

    <Step title="Connect your WhatsApp account">
      Open your browser to `http://localhost:3000` and scan the QR code with your WhatsApp mobile app:

      1. Open WhatsApp on your phone
      2. Tap Menu (⋮) or Settings and select **Linked Devices**
      3. Tap **Link a Device**
      4. Scan the QR code displayed in your browser

      <Note>
        You can also use pairing code authentication instead of QR code by visiting `/app/login-with-code?phone=628123456789`
      </Note>
    </Step>

    <Step title="Send your first message">
      Once connected, send a test message using the REST API:

      ```bash cURL
      curl -X POST http://localhost:3000/send/message \
        -H "Content-Type: application/json" \
        -d '{
          "phone": "628123456789@s.whatsapp.net",
          "message": "Hello from GOWA!"
        }'
      ```

      <Accordion title="Example response">
        ```json
        {
          "code": 200,
          "message": "Success",
          "data": {
            "message_id": "3EB0C0A6B4B8F8D7E9F1",
            "status": "success"
          }
        }
        ```
      </Accordion>
    </Step>

    <Step title="Explore advanced features">
      Now that you're connected, explore more capabilities:
      - [Send images, videos, and files](/guides/media-handling)
      - [Set up webhooks](/integrations/webhooks) to receive incoming messages
      - [Manage groups](/guides/group-management) and participants
      - [Integrate with Chatwoot](/integrations/chatwoot) for customer support
      - [Use MCP mode](/mcp/introduction) for AI agent integration
    </Step>
  </Steps>
</div>
